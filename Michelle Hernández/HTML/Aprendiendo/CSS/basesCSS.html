Ejemplo de mi tercer selector (ejemploSelectorTAG)

Este es otro estilo :)

CSS (Cascading Style Sheets) esta basado en JSON (JavaScript Object Notation) el cual debe tener una estructura llave-valor, en sí son reglas de estilo con selectores:
selector{
propiedad1: valor;
propiedad2: valor;
}

Selectores: forma de identificar elementos. La jerarquía esta basada en la prioridad según el tipo de elemento:

1) #id: vease ejemploSelectorID, se referencia en "style como #ejemploSelectorID", y se coloca el estilo dentro de llaves ({});

2) .clase: vease ejemploSelectorCLASS, un elemento puede tener varias clases sí son separas por un espacio " ".

3) !>etiqueta>:vease ejemploSelectorTAG, se referencia en "style como ejemploSelectorTAG {}".

4) !>etiqueta> !>etiqueta>: vease ejemploSelectorReglaCombinada, se especifica entre etiquetas.

5) Diseño GENERAL: si se desea tener todo el estilo igual en todo:
*{
color:red;
}

6) Sobre-escribir en librerías: sí se desea sobre-escribir usar "!important" en los selectores de >style> deseados.
.ejemplo{
color:green !important;
background: black !important;
}
